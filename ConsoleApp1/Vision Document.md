# Vision Document — Space Missions DB

## 1. Overview
**Space Missions DB** — реляційна база даних для зберігання структурованої інформації про космічні місії, космічні апарати, екіпаж, телеметрію, аномалії та наземні станції.

## 2. Purpose
Мета проєкту — створити централізовану систему для збереження, структурування та аналізу даних космічних місій, що забезпечує зручний доступ до інформації для операторів, аналітиків та інженерів.

## 3. Stakeholders
- **Mission Operators** — моніторинг місій.
- **Engineers and Technicians** — аналіз технічних параметрів та телеметрії.
- **Analysts** — робота з телеметрією та аномаліями.
- **Management** — контроль статусів місій.

## 4. Project Scope
### In scope:
- Зберігання даних про місії та їх компоненти.
- Телеметрія, аномалії, екіпаж, апарати, наземні станції.
- Зв’язки між сутностями та запити до БД.

### Out of scope:
- Реальний центр управління польотами.
- Інтерфейс користувача.
- Автоматизована обробка телеметрії.

## 5. High-Level Features
- Каталог місій
- Каталог космічних апаратів
- Історія екіпажу
- Логи телеметрії
- Логи аномалій
- База наземних станцій

## 6. Success Criteria
- Структура БД повністю нормалізована.
- Забезпечено референційну цілісність.
- Можливість виконання складних запитів (JOIN, GROUP BY).
- БД легко розгортається та заповнюється тестовими даними.
